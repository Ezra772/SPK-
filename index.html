Alur Metode Profile Matching

01

Hitung GAP

GAP = nilai alternatif - nilai ideal. Profil ideal semua kriteria bernilai 5.

02

Konversi Bobot GAP

Setiap GAP diubah menjadi bobot, misalnya GAP 0 bernilai 5 dan GAP -1 bernilai 4.

03

Hitung CF & SF

Core Factor berisi C2, C3, C5. Secondary Factor berisi C1, C4, C6.

04

Ranking

Nilai akhir = 60% Core Factor + 40% Secondary Factor, lalu diurutkan dari tertinggi.
